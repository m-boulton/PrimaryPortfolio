Coding Languages Learned

Initially learned HTML and CSS, decided to add some initial code onto github but didnt continue since was still in a learning stage, then moved on and learned Javascript vanilla, then added some jquery to my projects, at that point decided to learn the Reactjs library and Redux for state management but since the introduction of react hooks; have not needed to implement redux in any projects.
After learning these frontend languages decided to host projects on my own virtual private server, while looking into the deployment of these projects I decided to host all future code on my own Git server (instead of github) and use git hooks to deploy to production folders.
My Virtual Private Server is based on debian 10, have been able to use the debian client with ssh logins and putty, decided to use nginx as a proxy service and certbot certification keys for https web connections and smtp email services, all backend services are built in nodejs and used a variety of npm packages like Expressjs mongoose nodemailer dotenv and nodemon.
Since a database was needed I have been using the Mongodb.com collections Saas and have found it quite efficient.
