Coding Languages I've Learnt

Initially I learnt HTML and CSS, I decided to add some initial code onto github but didnt continue since I was still in a learning stage, then I moved on and learnt Javascript vanilla, then I added some jquery to my projects, at that point I decided to learn the Reactjs library and Redux for state management but since the introduction of react hooks I have not needed to implement redux in any projects.
After learning these frontend languages I decided to host my projects on my own virtual private server, while looking into the deployment of my code I started to realize I wanted to host all future code on my own Git server (instead of github) and use git hooks to deploy my code to production folders.
My vps is based on debian 10, I have been able to use the debian client with ssh logins and putty, I decided to use nginx as a proxy service and certbot for certification keys for https web connections and my smtp email services, all of my backend services are built in nodejs and I used a plethera of npm packages like Expressjs mongoose nodemailer dotenv and nodemon.
Since I needed a database to store data I have been using the mongodb.com collections Saas and have found it quite efficient for my use case.

Current Projects

Currently im building a reactjs based login page that allows an account login and a signup form for new users, im implementing bootstrap for styling and using JWT tokens stored in the local storage of the browser with a refresh token system for administration token management.
I will be using this login system for future projects that require user credentials, I know this is already done with authorization services like Oauth and authO but I wanted to build an authorization system so that I would be able to work with a token system in conjunction with a database.
